* npm init
package.json을 만든다.

* npm run test
package.json에 정의되어 있는 test 명령어 실행.

* npm install express
express를 설치한다.

* npm install morgan cookie-parser express-session
모듈을 동시에 여러개 설치.

* npm install --save-dev nodemon
개발용 패키지를 설치.

* npm install --global rimraf
전역설치.

* rimraf node_modules
설치한 로컬 모듈을 삭제.
